# Pmlatex

Upload PDF file created by `pmlatex.sty` and `pdflatex` to Medusa

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'pmlatex'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pmlatex

## Usage

See online document:

    $ pmlatex --help
